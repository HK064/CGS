# CGS

## 実行方法
Windows なら、用途に応じて以下のバッチファイルを実行する。<br>

### W-compile.bat
コンパイルする。<br>
引数なし（直接実行）だと停止。<br>
引数あり（呼び出し）だと、コンパイル成功 or 失敗 (0 or 1) を返す。<br>

### W-run.bat
実行する。<br>
引数なし（直接実行）だと停止。<br>
引数あり（呼び出し）だと停止せず終了。<br>

### W-compile-run.bat
コンパイルして、成功したら実行する。<br>

### W-javadoc.bat
javadoc を作る。<br>
